# MovieApp 🎬

A modern movie discovery application built with React, Bootstrap 5, and TMDB API.

## Features

- 🏠 **Home Page** - Display popular movies
- 🔍 **Search Functionality** - Real-time movie search
- 📄 **Movie Details** - Comprehensive movie information
- 📱 **Responsive Design** - Mobile-first approach
- 🎨 **Modern UI** - Clean and intuitive interface
- ⚡ **Fast Performance** - Optimized with Vite
- 🔄 **Pagination** - Browse through multiple pages

## Tech Stack

- React 18
- Bootstrap 5
- React Router DOM
- Axios
- TMDB API
- Vite

## Getting Started

### Prerequisites

- TMDB API Key
